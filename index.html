Conocimientos

Los cursos que tuve a lo largo de la carrera me han ayudado a desarrollar las bases de la programación que son transversales a los distintos lenguajes o tecnologías que hay disponibles. Con esto me refiero al pensamiento lógico, algoritmos, estructura de datos, y uso de memoria.
Luego, estudié de manera autodidacta valiendome de cursos online, leyendo documentación, y realizando proyectos, las siguientes tecnologías.

Back End

  • Hasta el momento mis desarrollos se centraron en NodeJs y las tecnologías que se suelen usar en conjunto como Express, Passport, Mongoose, Sequelize.
    Ahora también estoy aprendiendo TypeScript para hacer más robustos mis desarrollos, sigo profundizando en el uso de Node, y además estoy comenzando con Java.

Front End

  • En el front-end utilicé HTML y CSS (con la ayuda de Bootstrap y PUG), además de JavaScript para agregar algunas funcionalidades. Actualmente estoy aprendiendo React.

Bases de Datos

  • En cuanto a base de datos, tengo conocimientos en bases relacionales como MySQL y PostgreSQL, y no relacionales como MongoDB. En los proyectos realizados utilice el ORM Sequelize para las SQL y el ODM Mongoose para NoSQL.

Testing

  • Para los tests estoy utilizando el framework Jest, junto a SuperTest y Postman, realizando tests unitarios y de integración.
